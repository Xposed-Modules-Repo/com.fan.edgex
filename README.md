# EdgeX - Gesture Control

[English](./README_EN.md)

## 简介

EdgeX 是一个面向 Android 15+ 的 LSPosed/Xposed 模块，目标是把边缘手势、按键动作、冻结应用抽屉和一些高频系统操作整合到一起，向 Xposed Edge 致敬。

目前已实现的功能包括：

- 16 个边缘触发区域：左/右/上/下四边均支持三段细分区域，并提供整条边缘的低优先级兜底区域。
- 每个区域可分别配置单击、双击、长按，以及符合边缘方向的滑动手势。
- 支持子手势：可在主手势后继续配置停留、二段上/下/左/右滑动等链式动作。
- 硬件按键动作：支持音量上、音量下、电源键的单击、双击、长按。
- 可分配动作包括：返回、主页、多任务、展开通知栏、锁屏、截图、全局复制、杀死前台应用、清理后台应用、亮度调节、音量调节、启动指定应用、启动应用快捷方式、音乐控制、自定义 Shell 命令。
- Freezer 功能：冻结/解冻应用、保存冻结列表、从手势或按键呼出 Freezer Drawer，并支持一键重新冻结。
- Freezer Drawer 支持普通网格布局和弧形布局两种显示方式。
- 提供主题配色、自定义 RGB、手势调试区域显示、动作图标、更新检查等附加功能。

使用前请注意：

- 模块目前以 Android 15+ 为目标。
- LSPosed/Xposed 作用域需要至少包含 `android` 和 `com.android.systemui`。
- Freezer、清理后台、部分快捷方式读取和部分 Shell 命令依赖 root 权限或系统级 Hook 能力。
